# 🚢 Titanic Survival Prediction

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Machine Learning Model
- Logistic Regression

## Features Used
- Passenger Class
- Gender
- Age
- Fare
